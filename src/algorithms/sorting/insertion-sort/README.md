# Sắp xếp chèn

Insertion Sort (Sắp xếp chèn) là thuật toán hoạt động giống như cách bạn sắp xếp bài trên tay:

Chèn từng phần tử vào đúng vị trí trong phần đã được sắp xếp

Bắt đầu với phần tử đầu tiên (coi như đã sắp xếp)

Lần lượt lấy phần tử tiếp theo và chèn vào vị trí thích hợp

# Nhận xét và so sánh với các thuật toán Sắp xếp

Sắp xếp chèn là thuật toán sắp xếp đơn giản để xây dựng một mảng (danh sách) cuối cùng bằng việc thêm từng mục tại một thời điểm. Nó kém hiệu quả hơn nhiều trên các danh sách lớn so với các thuật toán nâng cao hơn như quicksort, heapsort hoặc merge sort.
